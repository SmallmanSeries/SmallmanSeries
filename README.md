## 小人系列工作室
一个小人掉下来了！　|　[A Smallman Fell From Sky!](README_English.md)
### 简介
小人系列工作室，简称小人系列，是一个基于架空世界观的游戏、动画工作室。当前仅有 冒险家岐哥 与 SdF隐姓埋名 两位成员。
### 历史
**2014年**，一个名为“打怪队”的虚构组织成立了。阿岐（当时的冒险家岐哥）在其中担任“队长”一职，受到“大王”的管辖。“打怪队”本质上是一个架空世界观创作组织，可以说它就是小人系列的雏形。

**2015年**，阿岐开始创作漫画。漫画的主角是一个脸很喜感的大头火柴人，这就是 小人 的原型。

**2016年**，阿岐与当时的 SdF隐姓埋名 合作创作了虚构游戏《打怪小游戏》，并首次使用了 小人系列 作为开发商的名称，以 小人 作为标志和吉祥物，这也标志着小人系列的正式诞生，史称“第一次成立”。

**2019年**，也就是小人系列的两名创始人进入初中的时候，小人系列逐渐开始正规化，其作品也开始成体系化。此时，小人系列正式确立了“基于架空世界观的游戏、动画工作室”这一基本定位，且开始致力于打造IP形象。

**2020年~2022年**，由于疫情的影响，小人系列进入了前所未有的“活跃期”。代号「小人历险记」、代号「小人星的高科技」等大型项目纷纷正式立项。

**2023年8月**，小人系列的第一个公开项目「MC探境者」正式立项，并在一年后发布。这是工作室的一个重大转折点，标志着其正式从几个人之间的自娱自乐变成了真正能产出内容的工作室，史称“第二次成立”。

**现在**，小人系列工作室的主要任务是Minecraft模组「边境旅者」的制作与立项筹备，同时持续维护代号「小人历险记」、代号「小人星的高科技」等大型项目。
### 位置
小人系列工作室位于奥根大洲洲府临春市，本质上就是岐哥他们的住宅。
### 作品（年份为立项时间）
#### 虚构作品
- **《出发吧打怪队》（2014）**：第一部虚构作品，以“打怪队”与来自时空隧道的怪物之间的对抗为中心，展开的奇幻世界观。
- **《神树世界》（2015）**：虚构的MC模组，其添加了100余个维度，还有各种为论战而生的神仙生物和神器，其中最弱的“神树怪”都有∞的血量，最强的“核弹风暴大王”甚至叠到了论天级别。
- **《小人漫画集》（2015）**：阿岐创作的虚构漫画集，首次使用了小人作为主角。
- **《打怪小游戏》（2016）**：首次以“小人系列”的名义创作的虚构游戏，拥有完整的世界观。玩家将扮演一名二维生物，在各个地点击杀各种怪物，并推进剧情，最终完成升维。
- **《死亡小人》（2017）**：由当时的员工 朱BUG（于2019年离职）创作的虚构漫画集，同样以小人为主角，以其脑洞大开的故事而闻名。
- **《方块冒险》（2019）**：将《打怪小游戏》重新包装设计的虚构游戏。
#### 正式作品
- **《集块小游戏》（2018）**：使用GameMaker制作的小游戏，其目标是收集足够多的从天上掉下来的方块。无限期停更。
- **《MC探境者》（2023）**：Minecraft边境之地科普视频系列。[链接↗](https://www.bilibili.com/video/BV1ARWve4E1k)
- **《细节里的MC》（2025）**：Minecraft冷知识科普系列。[链接↗](https://www.bilibili.com/video/BV1eYnmzZEFf)
- **《边境旅者》（？）**：Minecraft边境之地主题冒险模组。[链接↗](https://github.com/SmallmanSeries/FarLandsTraveler)
